# Introduction to Express package

- What is a framework? How is it different from library?

  Frameworks are code that someone else wrote, usually a lot people wrote because it's a lot more code that we can use instead of our app.
  The most important difference between a library and a framework is `Inversion of Control`. It means when you call a library, you are in control, but with a framework the control is invented: the framework calls you.
